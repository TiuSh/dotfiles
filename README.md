# TiuSh Dotfiles

## Install

* Clone this repo

`$ git clone git@github.com:TiuSh/dotfiles.git ~/.dotfiles`

* Execute install script

`$ ~/.dotfiles/install`

* Restart terminal and run post-install script

`$ ~/.dotfiles/post-install`
