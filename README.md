# TiuSh Dotfiles

## Install

* Clone this repo

`$ git clone git@github.com:TiuSh/dotfiles.git ~/.dotfiles`

* Execute install script

`$ ~/.dotfiles/install`

* Update Neovim plugins

`$ nvim +PlugInstall +UpdateRemotePlugins +qa`
